Mods

  • Rules
  • AddKingdomItems - Adds kingdom items like wagons, banners, flags, etc.
  • AffinityFixer - Replaces PVP or other useless PVE affinities. To use the Affinity Fixer, open skill window and right-click on the above-mentioned affinity and select 'Fix Affinity' and then relog.
  • Alchemy - Potions and Poisons can be created. Link to Alchemy Mod
  • Announcer - Server-wide player login announcement.
  • AshProduce - Items that have cooling ticks (Oven, forge, campfire) can produce ash.
  • Bagofholding - Adds Bag of Holding spell to exponentially increase container sizes.
  • BalanceMod - Allows command-line weirdos aficionados to retrieve their bank balance by typing /balance.
  • Bdew_server_mod_tools - Back end library for mods.
  • Betterdig - Keeps dirt from going into inventory when digging.
  • Betterfarm - Sow, farm and harvest multiple tiles at higher skill levels.
  • BetterGamemasters - Allows GMs unlimited inventory weight.
  • BoatMod - Small changes to make boats better.
  • Bountymod - Kill creatures for money and Karma. Allows player-bred creatures to be culled for rewards.
  • Bounty on burn - Burn corpses for money and Karma.
  • Bulkmod - Place hot items directly into BSBs.
  • Bulkseperated - Sorting containers like crates, FSBs, and BSBs.
  • CommonTweaks - Small tweaks, mostly for enchanting rift items for future use.
  • CookingMod - Fills cooking containers automatically (right-click and select Fill) must know recipe and have ingredients in inventory.
  • Creatureagemod - Age creatures to Mature quickly, slow down aging after.
  • Creaturemod - Adds new creatures and variants of existing creatures.
  • Cropmod - Prevents crops from weeding.
  • CustomWagons - Adds many new wagon model types.
  • Decorations - Adds various decorations to create/place.
  • Deedmod - Customizing deed configurations on the back end.
  • DisableFogSpiders - Does what it says. Fog spiders cause lag.
  • DiscordRelay - Connects kingdom chats to Discord (eventually).
  • DuskCombat (disabled for now) - Reworked combat system, will likely be implemented in future. Required for other mods and currently disabled.
  • EmptyTrashBin - Flint and steel on a trash heap will empty the heap immediately.
  • FireBurnTime - Examine fires to see how many minutes remaining on burn time.
  • Fixguards - Fixes issues with deed guards, backend mod.
  • HarvestHelper - Get exact season times with /seasons, alerts what is in season on join.
  • Hitchingpost - Create hitching post item, can hitch up to 4 animals to a post.
  • Hitchlimits - Allows control over what animals can be hitched to carts/wagons.
  • Httpserver - Internal web server mod for backend uses.
  • Highwayportals - Player craftable portals that allow teleporting between deeds on the same highway network.
  • Iconzz - Custom icons for various mods.
  • Inbreedwarning - Warns when breeding related animals.
  • InfiniteWideMineEntrances - No limit to mine entrance sizes, mine doors still work on each entrance.
  • LocationMod - Find your coordinates on the map at any time by typing /location in the chat.
  • MarkandRecall - Create runes from marble bricks to be able to mark and teleport. Distance relies on quality of the rune.
  • Meditatemod - Removes meditation restrictions and allows for easier meditation path following.
  • MightyMattockMod - Adds Mattock Pick, which lets you terraform on tile borders. Can flatten rock beneath dirt, chop trees, and harvest resources.
  • ModifyGestationPeriod - Decreases animal gestation, adds Decrease Gestation priest spell.
  • Movemod - Custom movement configurations - player speed, encumbrance, etc.
  • Movetocenter - Allows objects to move to center of tile.
  • NoMailCost - Removes mail costs.
  • PickMoreSprouts - More sprouts per "Pick" action.
  • SacrificeMod - Allows for chance of a strange bone when sacrificing a rare, supreme, or fantastic item in an altar (priest only) 1/20 chance.
  • Salvemod - Adds "power - " to the beginning of newly made healing cover names, so you no longer have to figure it out in your head.
  • Scriptrunner - Allows for backend scripting connections.
  • Servermap - Custom map configurations for backend (Live map at: https://terra-nullius.github.io/wurmmapgen/).
  • Serverpacks - Allows clients to download custom content from the server.
  • ServerTweaks - Various configuration changes for server settings.
  • SimpleConcrete - Create concrete with Clay + Rock Shards.
  • SinduskLibrary - Backend library for mods.
  • Spellcraft - Reworked spell system for more fun, adds several spells (see server settings).
  • Spellmod - Removed priest restrictions.
  • StableMaster - Convert your animal to a token to carry in inventory.
  • Startergear - Configurable starter gear for newbies.
  • Taxidermy - Create a taxidermy kit to stuff and mount your trophy kills.
  • Tentsleep - Sleep in your tent for a small sleep bonus!
  • Timerfix - Fixes various action timers.
  • Treasurehunting - Treasure maps that lead to buried treasure and some danger.
  • Upkeepcosts (removed for bug testing) - Allows for customized upkeep costs for deeds.
  • VotingRewards - Earn coin for voting for the server on Wurm-Unlimited.com (Link to Voting Page).
  • Windmill - Adds 3 different mills to bulk-create items like planks and bricks.
  • WyvernMod - A thousand little tweaks, added creatures like Titans and Wyverns, a lot of QoL options.
